**刚才，我们已经将各个簇分配到了细胞类型。然而，确认细胞类型是否正确分配是非常重要的。**

`FeaturePlot` 函数可以在 UMAP 图上显示每个基因的表达情况，因此我们可以用它来检查标记基因的表达情况。

这次我们将使用 CD14 作为单核细胞的标记，CD3E 作为 T 细胞的标记。请执行以下代码：

```r
FeaturePlot(pbmc, features = c("CD14", "CD3E"))
```

如果得到如下结果，则表示成功。

![image.png](https://raw.githubusercontent.com/aletolia/Pictures/main/202407082201042.png)

在被分配为单核细胞（monocytes）的簇中，可以观察到 CD14 的表达；在被分配为 T 细胞（T-cell）的簇中，可以观察到 CD3E 的表达。通过这种方式，实际绘制并检查各类标记基因的表达情况是非常重要的。
